# Network Scanning using Nmap

## Objective
To understand basic network scanning and reconnaissance techniques.

## Tools Used
- Kali Linux
- Nmap

## Steps Performed
- Installed Kali Linux in a virtual machine
- Performed basic Nmap scans on a test system
- Identified open ports and running services

## Key Learnings
- Understanding open ports and services
- Basics of network reconnaissance
- Importance of network security
